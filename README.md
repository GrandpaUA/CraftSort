# CraftSort

Sorts crafting station recipes by item stats — revival of the deprecated SortCraft mod.

**Supports:** Armor · Block · Physical DMG · Elemental DMG · HP · Stamina · Eitr · A→Z

## Installation
1. Install [BepInEx 5.4.x](https://thunderstore.io/c/valheim/p/denikson/BepInExPack_Valheim/)
2. Drop `CraftSort.dll` into `BepInEx/plugins/`

## Building from source

### Requirements
- Visual Studio 2022
- .NET Framework 4.8
- Valheim installed via Steam

### Setup
```bash
git clone https://github.com/ТВІЙ_НІК/CraftSort.git
cd CraftSort
# Set your Valheim path (or it defaults to the standard Steam path)
set VALHEIM_INSTALL=C:\Path\To\Valheim
```
Open `CraftSort.sln`, build → Done.

## Compatibility
Valheim 0.221.13+ (Unity 6 engine)
